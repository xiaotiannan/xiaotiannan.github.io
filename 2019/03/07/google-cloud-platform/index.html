google云打造下载速度20m的梯子

准备

  • google账户一个
  • 双币信用卡一个（我的是招行的）
  • 脑子一个

申请google账户

  • 关于如何申请google账户我不说了，可能会遇到坑。
    • 中国的手机接受不到验证码
    • 梯子打不开
    • 手机不能验证

      试用google cloud platform

  • google云链接
  • 点击页面中的免费试用
  • 如图中的一样公司地址之类的要填好，我填写的新加坡的地址-南洋理工学院的地址和邮编请自己搜索
  • 然后再填写双币信用卡信息。具体的请看各个银行的信息
  • 之后你的google 云的账户已经申请好了。（你们都申请多少钱？为什么我会是407.12美金）

创建VM实例（申请服务器）

  • 点击导航菜单进入-Compute Engine-VM实例-创建实例
  • 区域写：台湾 地区：asia-east1-c
  • 机器类别选择-微型（一个共享vCPU）
  • 个人喜好：启动磁盘选择Ubuntu 18.04
  • 防火墙勾选 ：允许HTTP流量 和 允许HTTPS流量
  • 下面还有一行字：管理,安全,磁盘，单独使用点击安全
    • 添加ssh密钥（如果你是mac可以在命令行中打cat ~/.ssh/id_rsa.pub）,然后将包含ssh-rsa开头的的一大段字母复制进密钥
    • 如果不添加也行，可以通过浏览器打开服务器的命令的
    • 创建固定的外部IP地址如图
  • 创建成功

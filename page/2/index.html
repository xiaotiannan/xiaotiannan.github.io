  • 由于最近减脂肪太多,需要调整,决定阅读这本书,会根据书中的实例,来进行自我挑战

健身入门

健身的基础知识

渐进的过程

  • 身体所承受的压力比它的习惯大,它就会去适应.从而变得更强壮.
  • 使用正确的练习,使用正确的技巧,这样才能向你的神经系统发出信号,告诉你的身体你想它达到的适应度.
  • 渐进式训练原则.

举重,阻力训练和健身

  • 举重

    • 奥林匹克举重
      • 抓举和挺举
    • 力量举重
      • 硬拉
      • 仰卧推举
      • 深蹲

  • 举重运动员的力量训练和健身训练的差异

    • 举重:使用的重量尽可能重,动作反复度较低
    • 健身:使用的重量较小,运作反复度较高

  • 健身训练如何有效

    • 肌肉维饿收缩遵循”不进则退”的原则
    • 能够举起的重量取决于三个因素
      • 能够征用的肌肉维的数量.
      • 单个纤维的强度
      • 所使用的技巧
    • 让健身最大化发展方式
      • 使用单次能举的最大重量的75%进行训练
      • 可以针对上半身进行8-12次反复训练
      • 针对腿部做12-15次训练
  • 健身与有氧耐力
    • 肌肉耐力
      • 运动过程中肌肉反复收缩,且为了完成动作征用最大数量的肌肉维的能力
      • 通过大量的肌肉收缩运动,增强肌肉耐力
    • 心肺耐力
      • 心脏,肺,循环系统将氧气运至肌肉维持运动的进行,并且带走产生的废品(乳酸)的能力
      • 通过大量的有氧运动来增强心肺能力

有氧运动和肌肉清晰度

  • 身体所能承受的有氧运动的压力并不是无限的.过多心肺训练会损害身体
    • 可能会减弱肌肉
    • 和整个身体的恢复能力
  • 塑造和发展肌肉的主要方式
    • 通过渐进式训练,举起铁家伙,并不是有氧训练.
  • 理想的体型
    • 宽阔的肩膀和背部,越往下越细,直到一个紧致的腰部,腿和驱干比例协调,肌肉发达,清晰,匀称,并且饱满直到关节部位,每个身体的部位都足够发达,包括三角肌后部,下背,腹部,前腿和小腿,肌肉有良好的清晰度和分离度.

如何增加强度?

  1. 增加训练重量
  2. 增加每一组中的反复次数.
  3. 减少组之间的休息时间
  4. 一次性不间断做两组或更多的组(超级组)

收益递减法则:你越资深,进步就越难,需要的训练强度就越大.

脖围,胸围,二头肌尺寸,前臂尺寸,手腕的尺寸,腰围,大腿的尺寸,小腿的尺寸,和体重

基础的训练原则

个体的需求:每个人的需求都是不同的

###循序渐进:如果不循序渐进就加大受伤的风险

* 反复
    * 上半身肌肉:8-12次的反复
    * 腿部肌肉:12-16次的反复.
* 练到力竭:不断的进行一组练习,如果最后不休息的话就不能多做一次反复
* 动作幅度:以最大的动作幅度来训练肌肉

在健身房受伤的原因: 

* 用了不合适的技巧(使用的重量过重或者不能完全控制重量)
* 没有进行恰当的拉伸和热身练习

拉伸练习:

* 站姿侧弯
    * 拉伸腹外斜肌和躯体测面的其他肌肉
* 站姿前弯
    * 拉伸腘绳肌和下背部
* 弓步压腿
    * 拉伸大腿内侧,腘绳肌.
 * 分腿坐姿前屈
    * 拉伸腘绳肌和下背部
 * 大腿内侧拉伸练习
    * 拉伸大腿内侧肌肉
 * 股四头肌拉伸:跪姿后仰
       * 拉伸大腿正面
 * 跨栏式拉伸 
        * 拉伸腘绳肌和大腿内测
   * 脊柱扭转练习
        *   增加躯干的旋转范围,拉伸大腿外侧肌肉
   * 悬垂拉伸练习
          * 拉伸脊柱和上半身

了解你的身体类型

* 外胚型:上半身短,四肢修长,手掌和脚掌又长又窄,同时脂肪存储非常少,狭小的胸部和肩部上通常依附着又长又细的肌肉组织
* 中胚型:宽阔的胸部,较长的躯干,结实的肌肉结构,以及极好的力量.
* 内胚型:柔软的肌肉组织,圆脸,短短的脖子,宽大的髋部,以及厚重的脂肪存储.

从低到高记作1-7

每种类型的身体类型都需要不同的训练计划:具体的计划需要在书中P141观看

健身者可以分为以下的肌肉群

* 后背---背阔肌的宽度和长度,背部厚度,中背部肌肉的发达度,下背部竖脊肌的发展
* 肩膀---大小和分满度,三角肌的三个头(前部,中部,后部)的发展,斜方肌
* 胸部---胸部上部和胸部下部,中胸的厚度,丰满的胸腔,躯干两侧的肌肉细节,前锯肌和肋间肌.
* 股二头肌---股二头肌的上部,下部,总长度,厚度.
* 股三头肌---股三头肌的三个头的发展,细节和分离度,块头和厚度
* 前臂---伸肌和屈肌的发展,肱肌与肘部的连结度
* 股四头肌和臂肌----肱四头肌全部四个头的发展,肱四头肌的分离度,大腿外侧的弧线,大腿内测的内收肌.
* 腘绳肌---股二头肌的饱和度和弧度,腘绳肌与股四头肌的分离度;
* 腹肌---腹肌的上部和下部,腰部两侧的腹外斜肌
* 小腿---浅层的腓肠肌和深层的比目鱼肌.

高级训练原则

* 增加训练强度
    * 基本的
        * 在更少的时间内完成相同的训练量
        * 在相同的时间内进行更多的训练, 
    * 增加训练负荷
         * 使用更大的重量进行训练
         * 缩减两组之间的休息时间,连续无间隔的进行两种或者三种不同的练习.
   * 强度技巧
        * 冲击原则
               * 使用比平时重的重量进行训练
               * 做更多的反复次数和组数,加快你的训练节奏,减少两组之间的休息时间,尝试新的动作
               * 打破以往的动作顺序
        * 强迫次数
            * 一种强迫进行额外次数的反复训练方法,就是在训练之时,让小伙伴帮助你继续做下去.
         * 局部次数
            * 当过于疲惫的时候,而不能进行全动作的反复时,可以做局部次数的反复.
        * 孤立训练
            *  针对某块肌肉,或者肌肉群,将其他的肌肉独立开来进行训练.
        * 负功训练
            * 举起重量:用的肌肉的收缩力,向心收缩.
            * 放下重量:肌肉得到伸展,称为离心收缩.
            * 在训练中慢慢的,有控制的将重量放下.用助力法将重量举起,缓慢放下.
        * 强迫负功训练
            * 放下的时候,请小伙伴(在确认安全情况下)压住的重量给予压迫,缓慢的放下.
        * 助力法
            * 借力突破自己的单块肌肉的极限,(借其他肌肉的力量)
        * 重负荷法
            * 在常规反复后,使用强迫次数,负功训练,强迫负功训练,达到完全力竭.
        * 强力训练原则
            * 不断的增加重量动作反复的次数变低.
        * 交叉租
            * 在整个训练过程中交叉插入强化的训练.
        * 优先原则
            * 在训练过程中,给予身体弱区或者先对落后的区域以优越性.* 
            * 应对弱区训练可以用以下方法
                * 休息一天,给特定身体部位立刻安排训练
                * 在训练开始的时候给予特定部位安排训练,而不在后期非常疲惫的时候
                * 选择有针对性的动作,正对特定部位.
                * 专注提高训练技巧,增加锻炼的效率和效果
                * 改变训练计划,针对落后的身体部位,进行额外的训练.
        * 超级组
            * 连续,无间隔地做两种练习的动作.或者做3隔组.
                * 可以针对同一个部位进行训练的两种练习.
                * 可以针对两个不同部位的超级训练.如背部和胸部
        * 递减法
            * 当即将无法进行某组训练的时候,减轻重量,可以肌肉继续下去.
            * 减去一些重量可以多一些反复.
        *  静力原则
            * 在两组练习之间1min休息的时候,要继续缩紧正在锻炼的肌肉,保持磊感.
        * 直觉原则
            * 感知自己的身体,理解它对训练的独特回应,并且依据它们对你的训练做出相应的调整.
        * 预疲劳原则
            * 肌肉组合锻炼的时候,先让几组肌肉疲劳效果差不多,如果不是,先孤立没有疲劳的肌肉做动作.使他们同时达到极限
        * 你来,我也来
            * 为了增加你的训练强度以及冲击你的肌肉.
            * 如做一组练习的时候,完成后交给训练的小伙伴,他做完了将重量交给你,这样哑铃不会在地上,记住: 无论多么的累,当你的伙伴将哑铃递来的时候一定要做.
        * 局部冲血法.
            * 使用一个较轻的重量,在动作路径上保持稳定,迫使肌肉在较长的周期内维持收缩.
        * 复合组
            * 为了冲击身体,让身体每个部位都做不同的练习.
        * "一个半"法
            * 在任意的一组锻炼中,完成一个完整的反复之后,再增加半个反复.然后轮流进行完整的反复直到整组结束.
        * 轮排制(21响)礼炮
            * 用动作的下半阶段进行一系列的半个反复.然后用动作的上半阶段进行一系列的半的反复.然后在进行一系列的完整反复.
        * 渐增负荷法
            * 对一个身体部位一周三练.
                * 第一次练,使用较多的组数和次数,但不要使用最重的重量
                * 第二次练增加重量,但是还要保存实力.
                * 第三次练使用非常重的重量,每组次数最多4-6次.
        * 弹震式训练
            * 猛的抬起训练,而不是以匀速将其举起.
                * 导致了阻力的变化
                * 征用了最大数量的白色块肌肉纤维,白色肌肉纤维比红色慢肌肉纤维更大(约22%)
                * 导致持续的力竭.

练就一流的身材,高级训练计划.

* 锻炼额外的肌肉块和逐渐形成肌肉形状.

身体部位练习

肩部肌肉

三角肌
* 由三个部分组成,厚实的,三角形的肌肉,位于锁骨和肩胛骨之间,向下延伸并一直插入上臂.
训练三角肌
* 基本动作
    * 平举:不能直接锻炼三角肌,但是能三角肌的三个头分离起来.
        * 抬起伸直的胳膊,向前平举,向后平举,向侧平举.
    * 推举
        * 将重量举到肩膀的高度,然后将杠铃举过头顶,可以向前或向后推举.
斜方肌
* 斜方肌是一块平的,三角形的肌肉,它从颈部开始向外,向下延伸,到达两块肩胛骨之间.
训练斜方肌
* 推举
* 前平举
* 直立划船

肩部练习

杠铃推举
练习目的:锻炼三角肌前部和三角肌侧部,坐姿比站姿要规范的多.
颈后推举
练习目的:锻炼三角肌的前部和侧部,任何的推举动作都会锻炼到肱三头肌.
提铃上举
练习目的:锻炼三角肌前部和三角肌侧部,并且使整个身体更加紧致有力.可以锻炼到斜方肌,臂部,背部,和肩部
借力推举
练习目的:通过运用比正常练习时更重的重量,或者通过在肌肉已经力竭的时候继续进行肩部推举动作,来进一步发展三角肌的力量
器械推举
练习目的:锻炼三角肌前部和三角肌侧部
哑铃推举
锻炼目的:锻炼三角肌前部和三角肌侧部,可以使动作达到更大幅度.
阿诺德推举
锻炼目的:锻炼三角肌的前头和侧头(推荐)
站姿侧平举
锻炼目的:能锻炼三角肌的外部,其次还会锻炼三角肌前部和三角肌后部的发展
拉力器单臂交叉侧平拉
锻炼目的:锻炼三角肌的外侧头,有主于三角肌前头和后头的锻炼
拉力器单臂侧平拉
锻炼目的:将练习动作的压力集中在三角肌的侧头.
坐姿拉力器单臂交叉侧平拉
锻炼目的:为了发展三角肌后部
反握哑铃过顶侧平举.
锻炼目的:锻炼三角肌侧部和后部,还帮助锻炼斜方肌.
哑铃前平举

腹部

腹部缺乏视觉效果的原因:
1 . 饮食控制的不够,所以在腹肌上有一层脂肪

2. 没有足够的孤立,全幅度并且高质量的训练.

最佳的腹部训练:慢速,有控制,最大动作幅度的练习,并且爱完全收缩的点上保持一段时间,以达到巅峰效果

  1. 腹肌上部: 卷腹–不要试图抬起整个腰部,使其离开地面.
  2. 腹肌上部和腹外斜肌:转体卷腹:
  3. 腹肌下部:反向卷腹
  4. 腹肌下部:悬垂卷腹

正则表达式高手攻略

发表于 2016-04-24   |   分类于 coding   |  

正则表达式简单模式

    • 匹配开头是hello,结尾是world的,中间可以任意字符.
    • . 元字符匹配除了换行符以外的字符
    • * 代表着数量指的是它前面的内容连续重复使用任意次数
    • 匹配以字幕h开头的单词–先是\b单词的开头,接着是\w+至少一个的字母或者数字,最后是单词的结束\b
    • \w 匹配字母或者数字或者下划线或者汉字
    • +匹配前面的内容一次或者更多次数,与*不同的是*可以匹配0次
    • 这个类似于准确匹配的意思,
    • \.字符转义,如果你想查找字符串中的. + *之类的话就需要用\来进行转义查询,\\这个就是表示\本身
  4. [abc]+—abcabc
    • 只匹配由abc 三个字母组成得到至少一个字符串
    • [ ]匹配中括号中的字符串,例
      • [0-9]是和\d含义完全一样的:一个数字
      • [a-z0-9A-Z]如果是英文环境下完全等用于\w

常用的元字符

代码 说明
. 匹配除换行符以外的任意字符
\w 匹配字母或数字或下划线或汉字
\s 匹配任意的空白符
\d 匹配数字
\b 匹配单词的开始或结束
^ 匹配字符串的开始
$ 匹配字符串的结束

常用的限定符

代码 说明
* 重复零次或更多次
+ 重复一次或更多次
? 重复零次或一次
{n} 重复n次
{n,} 重复n次或更多次
{n,m} 重复n到m次

正则表达式的正常模式

  1. \(?0\d{2}[) -]?\d{8}
    • 匹配几种格式的电话号码-(010)88888888 010-88888888 01088888888等
      • 分析:\(表示转义的(, ?表示它出现0次或者1次;之后接了一个0,后面跟着2个数字\d{2},然后是[) - ]?表示跟着)或-或空格中的一个出现1次或者0次,最后出现8个数字\d{8}.
    • ?表示前面的内容出现0次或者1次
    • 这个也能匹配到010)88888888 这样错误的格式要解决这样的问题需要用到分支条件

  2. 0\d{2}-\d{8}|0\d{3}-\d{7}–010-88888888 0817-8888888

    • 表示匹配到连字符分隔的电话号码
      • 一种是三位区号-010-88888888
      • 一种是四位区号0817-8888888
    • |是分支条件,指的是几种规则,如果满足任意一种斗应该完成匹配 有中文意思或的意思

  3. \(0\d{2}\)[- ]?\d{8}|0\d{2}[- ]?\d{8}—(010)-88888888 或010-8888888

    • \(转移(,接着是个0,然后匹配到2个数字,接着\)转移),然后匹配到-一次或者0次,最后接着8个数字

  4. \d{5}-\d{4}|\d{5}—-55555或88888-8888

    • 这个表达式用于匹配美国的邮政编码。美国邮编的规则是5位数字，或者用连字号间隔的9位数字。
    • 给出这个例子是因为它能说明一个问题：使用分枝条件时，要注意各个条件的顺序。如果你把它改成\d{5}|\d{5}-\d{4}的话，那么就只会匹配5位的邮编(以及9位邮编的前5位)。原因是匹配分枝条件时，将会从左到右地测试每个条件，如果满足了某个分枝的话，就不会去再管其它的条件了。

  5. ((2[0-4]\d|25[0-5]|[01]?\d\d?)\.){3}(2[0-4]\d|25[0-5]|[01]?\d\d?)——-255.255.255.255

    • 表示一个正确的IP地址
    • 因为ip地址中的每个数字都不能大于255,才写的这么多
    • 先看中括号里面的2[0-4]\d|25[0-5]|[01]?\d\d?
      • 先匹配一个2,然后加0-4中的一个,接着加数字一个—240
      • 或者匹配25,然后加[0-5]中的一个 —255
      • 先匹配[01]?表示0或者1一次或者0次,然后加一个\d数字,最后加个数字\d?匹配0次或者1次.
      • 这样就表示0-255都匹配到了.后面一个{3}表示重复3次,再后面的{}里面的内容是前面去掉.的内容,这样整个就表示0.0.0.0到255.255.255.255的所有IP地址了

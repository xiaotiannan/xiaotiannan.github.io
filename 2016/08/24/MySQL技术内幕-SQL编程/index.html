MySQL技术内幕-SQL编程 的读书笔记

1. SQL编程

  • 将SQL编程分为三个阶段
    • 面向过程化的SQL编程阶段
      • 没有处理关系模型的经验和基于集合的思想
      • 滥用各种工具(游标,临时表,动态SQL语句),并且自己意识不到自己正在引起破坏
    • 面向集合的SQL编程阶段
      • 知道运用SQL编程需要更多的东西
      • SQL是基于关系模型集合理论的强大基础的产物
    • 融合的SQL编程阶段
      • 有丰富的知识并对SQL有深入的理解
      • 具备了判断什么时候使用纯静态的SQL编程方法不能完成某些任务的能力

数据库的应用类型

  • OLTP:online transaction processing (联机事务处理)

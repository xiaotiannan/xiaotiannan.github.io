Mac上常用软件整理

发表于 2016-02-08   |   分类于 Mac   |     |  

MAC上的软件

因为自己把电脑格式化了，所以所以软件需要重新安装，下面是我一次安装的顺序，

  1. 百度云盘
    • 软件的上传下载，国内必须的。
  2. 欧陆词典
    • 付费的软件，以前买过正版，注册码后来再邮箱里面找到的
  3. 网易云音乐
    • 因为歌单功能强大，必须下载的软件。精准推荐
  4. 微信
    • 微信 for mac 软件，聊天软件
  5. Flux
    • 保护视力的软件，对于每天坐再电脑面前的很长时间的人很需要。
  6. QQ
    • 国内聊天软件
  7. WizNote
    • 为知笔记，付费的笔记系统。
  8. snip
    • 国内鹅厂出品，appstore里面下载不能滚动截图，从官网下载才能滚动截屏。
  9. Skim
    • PDF看书工具–轻量
  10. VirtualBox
    • 免费的虚拟机系统，以后coding的时候前提之一
  11. Firefox
    • 火狐浏览器，国内可以下载
  12. Keka
    • 开源的解压缩软件，支持大部分解压缩
  13. Thunder
    • 迅雷下载软件，虽然不常用但是还是下载下来
  14. LICEcap
    • 制作GIF图片工具，开源
  15. Xcode
    • 再appstore里面下载
  16. Tickeys
    • 打字软件，可以让软件发出机械键盘的声音
因为需要下载chrome，想从官网上面下载，所以先下载翻墙软件。
  1. XX-Net
    • 通过配置，google开发，能翻墙，免费，快速，具体的请自己查看，先配合firefox翻墙
  2. ShadowsocksX
    • 因为以后需要自己搭梯子，所以先下载
  3. google chrome
    • 因为已经搭好梯子了，所以可以下载这个了。

如何阅读一本书笔记

发表于 2016-11-06   |   分类于 read book   |     |  

阅读的目标:

  • 为获得资讯而读:能增加资讯却不能增加理解力
  • 为求的理解而读:理解更多的事情
    • 开始时不相等的理解程度
    • 阅读的人一定要把不相等的理解力克服到一定程度内.
  • 阅读就是学习:指导型的学习,以及自我发现型学习之间的差异
    • 指导型学习称为”辅助的自我发现学习”
    • 非辅助型的自我学习-是阅读自我和世界的学习

  • 阅读的艺术包括了所有非辅助型自我发现学习的技巧

    • 敏锐的观察
    • 灵敏可靠的记忆
    • 想象的空间
    • 训练有素的分析

  • 强调阅读的重要性,将倾听作为第二.

    • 倾听是从一位出现在你面前学习的老师学习
    • 而阅读却跟一位缺席的老师学习.

  • 初学者的无知在于未学,而学者的无知在于学后

自我要求的读者

主动阅读基础:一个阅读者要提出的四个基本问题

  • 核心: 在阅读过程中提出问题来–在阅读过程中必须自己尝试去回答问题

  • 整体来说,这本书到底谈了什么?

    • 找出书本的主题作者如何依次发展这个主题的,如何逐步从核心主题中分解出从属的关键议题来的?
  • 作者细部说了什么?怎么说的?
  • 这本书说的有道题吗?是全部的有道理?还是部分有道理?
  • 这本书和你有什么关系?

如何让一本好书真正属于你

  • 读书做笔记不可缺少

    • 会让你保持清醒
    • 思考用语言表达出来
    • 帮助把你的感想写出来,帮助记住作者的思想

  • 做笔记的方法

    • 画底线—在主要重点,或重要又有力量饿句子下画线
    • 在画底线外的栏外再加画一道线
    • 在空白处做星号或者其他符号
    • 在空白处编号
    • 在空白处记下其他的页码–强调作者在书中其他地方有同样的观点
    • 将关键字或句子圈出来
    • 在书页空白处做笔记

  • 做笔记的三种方法

    • 结构笔记(structural note-marking)
      • 这是什么样的一本书
      • 整本书在谈论什么?
      • 作者借着怎么样的整体架构,来发展他的观点或陈述他对这个主题的理解
      • 写下这些问题的答案,可以在书的目录页.(重点是全书的架构)
    • 概念笔记(conceptual note-marking)
      • 读完,就可以回答关于这本书的准确性和意义的问题.
      • 这些概念是作者的观点,当读的越深就会出现自己的观点
    • 辩证笔记(dialectical note making) –针对一场讨论情景的笔记
      • 从好多本书中摘要出来

四种阅读层次

  • 基础阅读(elementary reading)
    • 认出页面上的文字
  • 检视阅读(inspectional reading)
    • 强调在规定的时间内阅读完一项阅读的任务
    • 在规定时间内抓住一本书籍的重点
  • 分析阅读(analytical reading )
    • 全盘的阅读,完整的阅读,或优质的阅读.咀嚼和消化一本书
  • 主题阅读(syntopical reading)也可以说比较阅读(comparative reading)
    • 读一系列的书籍

学习阅读的阶段(基础阅读)

这四个阅读阶段都属于基础阅读

  • 阅读准备阶段(reading readiness) 学前教育
    • 身体方面的准备
      • 良好的视力
      • 良好的听力
    • 智力方面的准备
      • 认知能力
    • 语言上的准备
      • 口齿清晰能说出正确的句子
    • 个人的准备
      • 能与其他孩童一起学习的能力
      • 保持注意力
      • 服从

  • 孩子会阅读简单的读物,认字 (一年级的学生)

    • 懂得一些符号的意义

  • 快速建立字汇的能力(四年级学生)

    • 所用的方法是从上下文所提供的线索,找出不熟悉的字眼
    • 在这个阶段,孩子会学会不同目标与不同领域的阅读法

  • 精炼和增进前面所学的技巧(终其一生学习) 九年级学生

    • 学生开始消化他的阅读经验
      • 从一本书所提出的观点转换到另一个观点
      • 在同一主题上,对不同读者所提出的观点做比较

检视阅读

  • 检视阅读一:有系统的阅读和粗读
    • 拿到一本书判断是否有价值
    • 想挖掘所有的东西,但时间有限

  • 略读”skim”或粗读(pre-read)

    • 先看书名,然后如果有序就先看序
    • 研究目录页
    • 书中有索引也要检阅一下
    • 读出版者的介绍
    • 挑选几个看起来与主题相关的篇章来看

    • 把书打开,东翻翻西翻翻,读几个片段,但是不要太多(不该错过最后2-3页)

    • 在花最多不超过一小时的时间里,略读过这本书了

      • 判断书本是否有值得你挖掘的内容
      • 脑中清楚这本书应该归为哪类.以后可以作为参考

  • 检视阅读二:粗浅的阅读

    • 头一次面对一本难读的书的时候,从头到尾先读完一遍,遇到不懂的地方不要停下来查询或思索

  • 阅读速度

    • 不止要能读的快,还要能用不同的速度来阅读
    • 知道什么时候用什么样的速度是恰当的.

阅读的第三个层次:分析阅读

分析阅读的第一个阶段:找出一本书在谈什么

  • 依照书的种类与主题来分类
    • 第一个规则:一定要知道自己在读哪一类的书,而且知道的越早越好,最好在你阅读之前就知道.
    • 主要的分类法
      • 虚构的小说类
      • 传到知识,说明性的论说类
        • 实用性作品
        • 理论性作品
  • 透视一本书
    • 第二个规则:使用最简短的文字说明整本书在谈论什么
      • 注意整本书的整体性
    • 第三个规则:将主要的部分按照顺序与关联性列举出来,将全书的大纲列举出来,并将各个部分的打个列举出来
-   驾驭复杂的内容:为一本书拟写大纲技巧
    +   举例
    +   作者将全书分成五个部分
        *   第一个部分谈论的是什么
            -   第一个主要的部分又分为三个段落
                +   第一个段落为X
                    +   在第一部分的第一阶段作者有4个重点
                    *  第一个重点是A 
                    *  第一个重点是B
                    *  第一个重点是C
                    *   第一个重点是D
                +   第一个段落为Y
                +   第一个段落为Z
        *   第二部分谈的是什么?
        *   第三部分谈的是别的事情
        *   第四部分则是另外的观点
        *   第五部分又是一些其他的事.
         这个大纲只是一个公式,需要灵活运用  

-  第四个规则:确定作者想要解决的问题

分析阅读的第二个阶段:诠释一本书的内容

  • 第五个规则:诠释作者的关键字,与她达成共识

    • 找出关键字
      • 作者开诚布公的强调某些特定的字
      • 作者与其他作者争执的某个用于就是重要的字
    • 每一个知识领域都有独特的专门用于(technical vocabulary)
    • 找出字义
      • 利用上下文自己已经了解的所有字句,来推敲出你所不了解的那个字义
    • 建议:在读一本书的时候,找出重要的关键词,确认每个字不同意义的转换,并与作者找出共通的词义,这样就会对一本书的理解力就会增加.

  • 第六个规则:由最重要的句子中,抓住作者的重要主旨

  • 第七个规则:知道作者在论述什么,从内容中找出相关句子,再重新架构出来
  • 第八个规则:确定作者已经解决了哪些问题,还有哪些没有解决,再判断哪些是作者知道他没解决的问题

分析阅读的第三阶段:像沟通一样的评论一本书

  • 智慧礼节的一般规则

    • 第九个规则:除非你已经完成了大纲架构,也能诠释整本书了,否则不要轻易的批判
    • 第十个规则不要争抢好胜,非辩不可以
    • 第十一个规则在说出评论之前,要证明自己区别得出真正的知识和个人观点的不同

  • 批评观点的特别标准

    • 第十二个规则:证明作者的知识不足
      • 缺少某些相关想要解决的问题相关知识
    • 第十三个规则证明作者的知识错误
    • 第十四个规则证明作者的不符合逻辑
    • 第十五个规则:证明作者的分析与理由不完整

阅读不同读物的方法

如何阅读实用型的书

  • 重要:任何实用性的书都不能解决该书所关心的实际问题

  • 两种实用性的书

    • 说明规则用的书籍
    • 阐释形成规则的原理

  • 读一本书的要提出四个问题的变化

    • 这本书在谈论什么?并且做出大纲架构.
    • 了解作者想做什么?了解他这么做的目的
    • 内容真实吗?
    • 这本书与我何干?看完后需要采取行动

如何阅读想象文字

  • 不要抗拒想象文字带给你的影响力
    • 在想象文学中,不要去找共识,主旨或论述
    • 不要用适用传递知识与真理一致的标准来批评小说

  • 架构性规则

    • 将想象文学作品分类
      • 抒情的:表达个人情感的经验为主
      • 小说与戏剧:许多角色,彼此,产生互动与反应,以及情感的变化
    • 抓住整本书的大意
    • 发现整本书各个部分是如何架构起来的

  • 诠释性,诠释规则

    • 小说的要素,插曲,事件角色与他们的思想感觉及行动
    • 共识与主旨:小说的要素与整个表现的场景或背景有关
    • 对角色都熟悉了,加入这个想象的世界,随着他们呼吸
  • 阅读批判性规则:在你衷心感谢作者试着为你创造的经验之前,不要批判一本有想象的作品

如何阅读故事,戏剧和诗的一些建议

  • 阅读故事书
    • 快读,并且全身心地读
    • 两种故事书的差异
      • 满足我们对个人特殊潜意识需求的小说
      • 满足大多数人潜意识需求小说(伟大的作品)

  • 阅读抒情诗

    • 不论你觉得懂不懂,都要一口气读完,不要停
    • 重读一边(大声读出来)
    • 阅读这类书需要知道一些关于作者的背景资料

  • 阅读希腊悲剧阅读建议

    • 悲剧的精髓在时间
    • 在读悲剧任务的台词时,你要想象这个是出自一个大牛的口中

如何阅读历史书

  • 为了追求历史的真相,我们必须从更多不同的角度来观察才行

  • 阅读历史时,不只要关心在过去的某一时间,地点真正发生的事情.

  • 历史书提问题

    • 弄清楚这本书在谈什么?没有谈什么?在什么限定的范围的主题
    • 哪点对他很重要?
    • 这与我何干?

##主题阅读

主题阅读中的准备

  • 对要研究的主题,设计一份实验性的书目,(参考豆瓣书单,专家的建议,系列书单,图书馆的目录,过来人的经验)
  • 检视阅读这本书单上面的书籍,确定哪些书是与你主题有关的.
    • 检视书单上的所有的书籍,会帮你简化书单到合理的程度.(节约时间)
    • 熟练检视阅读会花很短的时间,确认书籍是否对主题或者是想要研究的问题重不重要.(节约时间)

主题阅读的五个步骤

  1. 找出相关的章节(在主题阅读中,你所关心的的主题才是基础的重点,而不是你要阅读的书籍)
    • 将书籍整体检视一遍,找出与主题的相关的章节
  2. 带引作者与你达成共识(自己创造出中立的词汇)
  3. 理清问题:列出一连串的问题,让作者回答,
    • 有的作者一个问题回答不了,有的作者是可以回答的,具体的书本中寻找
  4. 界定主要和次要议题:
    • 将作者针对各个问题的不同意见整理在各个议题的旁边
  5. 分析这些讨论,将问题和议题按照顺序排列,来突显主题

阅读的心智与成长

  • 阅读时越主动,就读的越好
    • 不只要读的好,还要有能力分辨出哪些书能帮助你增进阅读能力
  • 好书给我们的帮助
    • 当成功阅读一本难度的好书后,你的阅读技巧会增加
    • 一本好书能教会你了解这个世界以及你自己,
      • 更懂得如何读的好
      • 更加懂得生命
      • 变得更加有智慧,而只是更加有知识
      • 会成为智者,对人类生命中永恒的真理有更加深刻的体认.
  • 书的金字塔
    • 第一类书籍:对阅读技巧毫无帮助的书籍,很多书只能娱乐和消遣(占百分之99)
      • 有趣的资讯非常多,不要想在里面学到任何重要的东西,有时候扫一眼就够了
    • 第二类书籍:可以让你学习的书—–学习如何阅读,如何生活,(可能百分之一,千分之一)
      • 这类书籍值得做分析阅读,可以常翻翻温故下知识点就好了
      • 在阅读这类书中心智会与书中的经验合二为一,增长你的理解力,心智.
    • 第三类书籍:读的再通,也不可能尽其究竟(有可能不到100种)
      • 如何辨别这类书籍?
        • 尽最大的努力分析阅读完,心中会有疑惑,有的还没有弄清楚
        • 即使你再重新看一边,还是没办法指出问题出在哪?你的疑惑在哪产生的.
      • 每次的翻阅这类书籍,总看到新的事物(以前读书的时候没看过的东西)
        • 这是会随你一起成长的书籍
